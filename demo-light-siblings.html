Shadow role — light DOM label & description siblings

The ARIA role lives on an inner shadow DOM element. The label and description are plain light DOM siblings — no slots, no shadow markup for them. Each custom element reads its host's aria-labelledby and aria-describedby attributes, resolves the referenced elements in the document, and wires them to the inner role element using Element.ariaLabelledByElements and Element.ariaDescribedByElements (element reference API, works cross-root). The combobox additionally uses Element.ariaActiveDescendantElement for its active option.
