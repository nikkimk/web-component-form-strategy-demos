What to try

  • Textfield: Tab to the field and type; the host keeps role="textbox". The inner input is aria-hidden.
  • Checkbox: Click or press Space; the host toggles aria-checked.
  • Progress bar: Shadow label and description; host owns role="progressbar" and updates aria-valuenow (not in tab order).
  • Inspect the log panels for host vs internals element refs.
