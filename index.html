Form fields strategy demos

Interactive PoCs for Q2 (role on the host) and Q3 (how label and help connect across Shadow DOM and Light DOM). Each page shows a textfield, checkbox, progress bar, and combobox using the same label wiring pattern.

Project README · Open in StackBlitz

Role on the host via ElementInternals

internals.role sets the ARIA role on the custom element host. Label and help wire through the host or internals depending on where they live.

  • Light DOM label and help — all four controls
  • Shadow DOM label and help — all four controls
  • Slotted label and help — all four controls

Role on an inner shadow DOM element

The ARIA role lives on a native <input> (textfield, checkbox) or a <div role="..."> (progressbar, combobox) inside the shadow root. ariaLabelledByElements and ariaDescribedByElements are set directly on that shadow element. No ElementInternals is used for ARIA wiring.

  • Shadow role — light DOM label and help
  • Shadow role — shadow DOM label and help
