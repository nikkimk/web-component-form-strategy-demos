Light DOM label and help text

Label and help text are Light DOM elements referenced on the host with ariaLabelledByElements and ariaDescribedByElements. The Shadow DOM listbox uses ElementInternals.ariaControlsElements. Options are slotted from Light DOM; the host uses aria-activedescendant for the active option.
