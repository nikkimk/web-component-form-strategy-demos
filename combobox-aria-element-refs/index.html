Combobox ARIA element reference demos

Interactive combobox custom elements. The Shadow DOM listbox is linked via ElementInternals.ariaControlsElements. Listbox options are slotted from Light DOM; the host uses aria-activedescendant for the active option. Shadow label/help refs also use ElementInternals; Light DOM label/help refs use the host.
