Shadow DOM label and help text

The combobox host carries role="combobox". Shadow DOM label and help are referenced through ElementInternals.ariaLabelledByElements and ElementInternals.ariaDescribedByElements. The listbox shell is linked with ElementInternals.ariaControlsElements. Options are slotted from Light DOM; the host uses aria-activedescendant for the active option.
