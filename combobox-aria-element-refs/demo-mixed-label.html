Mixed Light DOM and Shadow DOM associations

Light DOM label/help refs are set on the host; Shadow DOM supplements and the listbox use ElementInternals (ariaLabelledByElements, ariaDescribedByElements, ariaControlsElements). Options are slotted from Light DOM; the host uses aria-activedescendant for the active option.
