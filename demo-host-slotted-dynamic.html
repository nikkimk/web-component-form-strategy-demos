Slotted label and description — dynamic ID refs

Label and description are projected through named slots. On every slotchange, the component re-collects assigned nodes, assigns stable IDs when missing, and updates host.ariaLabelledByElements / host.ariaDescribedByElements (or aria-labelledby / aria-describedby in fallback mode).
